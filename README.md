# 핏밀리 - 가족과 함께 운동하며 건강을 나누는 소통형 헬스케어 앱

> **Fit**ness + Fa**mily** -> **Fitmily**

> **핏밀리(Fitmily)**는 가족 간 건강한 소통을 중심으로 한 웰니스 앱입니다.
멀리 떨어져 있어도, 함께 살고 있어도 서로의 건강을 응원하며
산책 위치 공유, 운동 목표 달성률 확인, 가족 챌린지 등을 통해
**"가족 모두가 함께 건강해지는 경험"**을 제공합니다.

#### 💡 기획 배경
현대 사회에서는 가족 간의 물리적 거리뿐만 아니라 심리적 거리도 점점 멀어지고 있습니다.
부모님의 건강이 걱정되지만, 실제로 어떤 운동을 하고 계신지, 얼마나 활동적인지 알기 어렵습니다.

“엄마는 요즘 어떤 운동을 하실까?”
“아빠는 오늘도 동네를 산책하셨을까?”

핏밀리는 이러한 궁금증과 걱정을 해소하고, 가족 간의 대화를 건강이라는 주제로 자연스럽게 이어주는 플랫폼입니다.
함께 걷고, 함께 응원하며, 함께 기록하세요.
***
#### 🎯 목표 및 주요 기능

###### 1. 가족 간 운동 & 산책 데이터 공유
    - 실시간 **산책 위치 공유**
    - 운동 현황 및 목표 진행률 **피트니스 링 시각화**
    - **캘린더 기반 달성률** 표시로 가족 전체 운동 상황 한눈에 확인

###### 2. 소통 중심의 헬스 게이미피케이션
    - **가족 산책 챌린지** 생성 및 달성
    - **단체 채팅방**을 통한 실시간 소통
    - **콕 찌르기** 기능으로 가족 간 운동 독려 및 응원

##### 👤 회원 기능
✅ 회원가입
이메일 인증, ID 중복 확인 하여 회원가입

✅ 로그인 / 로그아웃
기본 폼 인증으로 로그인 / 로그아웃

##### 홈 화면
✅ 패밀리 그룹 생성
패밀리의 건강상태를 한 눈에 대시보드로 확인 가능

✅ 산책 챌린지
가족 챌린지 달성 순위 등 확인 가능

✅ 현재 날씨 확인
현재 위치의 날씨 확인 가능

##### 👨‍👩‍👧‍👦 패밀리 기능
✅ 패밀리 그룹 생성
패밀리 그룹을 생성

✅ 패밀리 초대 및 가입
패밀리장은 그룹코드로 패밀리를 초대

##### 🏃‍♂️ 나의 운동 데이터
✅ 개인 건강 정보 등록
마이페이지에서 나의 건강상태 등록

✅ 개인 운동 목표 등록
마이페이지에서 나의 운동 목표 등록

✅ 개인 운동 기록 등록
마이페이지에서 나의 운동 기록 등록

✅ 개인 운동 기록 현황
마이페이지에서 나의 운동 기록 현황 확인 가능

##### 📊 가족 운동 캘린더
✅ 가족 운동 기록 캘린더 시각화
가족 운동 기록을 캘린더로 시각화하여 보여줌

##### 🌿 산책 기능
✅ 실시간 산책 공유
실시간 산책 하고 있는 패밀리 위치 공유

✅ 산책 기록 관리
패밀리가 종료한 산책 루트와 거리, 시간 등 상세 확인 가능

✅ 주간 산책 챌린지
패밀리의 주간 목표 산책 거리 생성 후 가족이 해당 거리를 같이 산책

##### 📣 가족 소통 기능
✅ 단체 채팅
건강을 주제로 가족 간 채팅 가능

✅ 콕 찌르기 기능
"김아빠"님이 "김엄마"님에게 산책을 장려했어요!"
